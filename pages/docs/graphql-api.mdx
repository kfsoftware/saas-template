After the client is created, we can use it to make requests to the GraphQL API.

## Create TODO

```graphql
mutation createTodo($tenantId: String!, $input: CreateTodoInput!) {
  createTodo(tenantId: $tenantId, input: $input) {
    id
    status
    text
  }
}
```

## Get all TODOs

```graphql
query getTodos($limit: Int!, $offset: Int!, $tenantId: String!) {
  todos(limit: $limit, offset: $offset, tenantId: $tenantId) {
    id
    status
    text
  }
}
```
